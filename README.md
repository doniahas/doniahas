## Selected Projects

### Alert Hub – Microservices Monitoring System

A microservices-based monitoring and notification system developed as part of a Java Backend & DevOps program.

The system collects task data from platforms such as Jira, ClickUp, and GitHub, processes events using Kafka, and sends alerts to managers through Email/SMS.

**Technologies:** Java, Spring Boot, Kafka, MySQL, MongoDB, Docker, Kubernetes, Jenkins

---

### Dormify – Dormitory Allocation Management System *(Ongoing)*

A full-stack system currently being developed for managing student dormitory allocation processes.

The system is designed to support student data import, dormitory and room management, role-based access, and allocation workflows. The project is being developed as a real-world academic system with a React frontend, Django backend, PostgreSQL database, and Docker-based development environment.

**Technologies:** React, Django, PostgreSQL, Docker, REST API

---

### Customer Churn Analysis

A data analysis and machine learning project based on the Telco customer churn dataset.

The project included data cleaning, exploratory data analysis, statistical testing, and building a kNN classification model to identify factors related to customer churn.

**Technologies:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

---

### Meetings App – Django Web Application

A Django-based web application connected to a relational database.

The system displays complex SQL query results, supports validated data entry through forms, and provides risk analysis and statistics based on stored records.

**Technologies:** Python, Django, SQL, HTML/CSS
